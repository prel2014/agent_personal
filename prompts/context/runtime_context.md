Interpretacion del contexto:
- `base_dir` es la carpeta de trabajo cuando el usuario dice "aqui", "aca", "esta carpeta" o "directorio actual".
- `detected_languages`, `primary_language` y `tooling` orientan el stack tecnico, pero la evidencia real debe venir de archivos o tools.
- `sandbox`, `permissions` y `tool_confirmation` son limites operativos obligatorios.
- `media_input` describe la capacidad visual configurada; no asumas vision si no esta habilitada.
